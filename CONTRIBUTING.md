# Contributing to SEC Filing Data Extraction System

Thank you for your interest in contributing to this project! This is a portfolio project designed to showcase data extraction and AI-powered financial analysis.

## How to Contribute

1. **Fork the repository** to your own GitHub account.
2. **Create a new branch** for your feature or bug fix:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Commit your changes** with clear and descriptive commit messages.
4. **Push your branch** to your fork.
5. **Open a Pull Request** describing your changes.

## Code Standards

- **Python**: Follow PEP 8 style guidelines. Ensure all new functions have type hints and docstrings.
- **Frontend**: Follow modern React best practices and keep components modular.
- **Tests**: If you add new logic, please include a script in the `backend/tests/` directory to verify the behavior.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on the GitHub repository with detailed information.
